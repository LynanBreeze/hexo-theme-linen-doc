By default, Markdown displays images one per line.

![](/img/duy-le-duc-GUY2f9csQbc-unsplash.jpg)

The Linen theme introduces a new "image container" feature, supporting the following image layout styles. Please click the **Switch to Raw Markdown Content** button on this page and refer to the page's source code to implement these image layouts.
